title: Overview
description: The agent runtime is where instructions, messages, tools, and events meet.
---

Agents are the central runtime primitive in Fragola.

Each agent owns:

- the current conversation state,
- its registered tools,
- its event handlers,
- its local store,
- and a live `context` surface for advanced integrations.

This section focuses on how to create agents, inspect their state, and shape their behavior.
