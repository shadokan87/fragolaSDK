Use the reference section when you already understand the concepts and need a compact lookup for the public runtime surfaces.

The pages here are organized by the main public entry points:

- [`Fragola`](/reference/fragola) for the client instance and shared runtime.
- [`Agent`](/reference/agent) for the execution surface.
- [`AgentContext`](/reference/agent-context) for hooks, tools, and event handlers.
- [`Store`](/reference/store) for mutable runtime state.
- [`State utils`](/reference/state-utils) for message inspection helpers.
- [`Errors`](/reference/errors) for the SDK-specific error classes you may want to catch.

If you are still learning how the pieces fit together, start in the Guides sections first and come back here when you need exact method surfaces.
